QRunInfo
==============================
        QRunInfo 是一些从Qt Installer Framework中提取出来的和自己封装的一些系统的信息获取和判断操作的封装。

================================
        环境：Qt 5.0+ <br\>
        系统:win,linux,mac、、、
    
=========================================
##简单介绍：
        KDRunOnceChecker : 检查程序是否是在运行中 ,或者文件有没有被锁定<br\>
        QRunInfo : 获取系统内存信息，检索正在运行的进程，杀死进程，判断是否是本地文件 <br\>
        AdminAuthorization : 判断当前程序是否是在管理员模式下运行，和以管理员程序开启启动程序 <br\>
        这是api的简单介绍：http://www.dushibaiyu.com/2014/10/qruninfo-api-smple.html
